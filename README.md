# JacaHello

My first gem for learning purposes.

It just says hello, that's all :smiley:

## Installation

Install it:

    $ gem install jaca_hello

## Usage

In the console open Interactive Ruby Shell...

    irb

...and then:

    > require 'jaca_hello'
    > JacaHello.hello
    
    Hello world!
